# API Gateway & Analytics Platform

A MERN-based API Gateway platform where users can register APIs, generate proxy URLs, apply rate limiting, and view request analytics.

## Current Progress

- Backend setup completed
- MongoDB connected
- User model created
- Register API implemented
- Password hashing using bcrypt
- User data saved in MongoDB

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- bcrypt
- JWT
- React
- Redis

## Upcoming Features

- Login API
- JWT authentication
- API registration
- Proxy URL generation
- Redis rate limiting
- Analytics dashboard
